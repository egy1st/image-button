Overview

DC Image Button
 Dynamic Components Image Button helps you personalize your program by incorporating your own buttons and icons. It gives you full control over your form design and elements, and even supports entire themes and moving images. Whether you’re writing a new application or rewriting an old one, DC Image Button can help you save time while giving your program a creative edge.

Interactive images
 With DC Image Button, your images can be programmed to change according to your user’s mouse actions. You can set them to change color, emboss or engrave, apply shadows, or even display another image when you click or mouse over. For database applications, you can also use this feature to create query forms.

Consistent design
 DC Image Button applies your design preferences to the entire form, instead of defining each element. This allows you to design your own theme, or create multiple themes for a more varied interface. You can also change individual GUI elements such as icons, buttons, and pictures.

Compatible with COM applications
 DC Image Button works with any COM-based application such as Visual C++, Borland C++, Delphi, and Visual Basic. Because it uses simple syntax, you can easily spot them when you scan your code for errors.

Simple installation and setup
 Installing DC Image Button takes less than a minute on Windows 98, NT, 2000, and XP. The program saves the file “DC_Image Button10.dll” at C:\Windows\system32, unless you specify another path. You can set it to launch when you open your project by adding the DLL file as a reference from the Project menu.
